## New API Endpoints

### Discover Vulnerabilities
```bash
POST /api/genai/discover
```
Returns 5 LLM-discovered vulnerabilities with:
- Attack mechanism
- Evasion strategy  
- Expected impact
- Severity rating

### Generate AI Attack
```bash
POST /api/genai/generate-attack
{
  "attack_type": "genai_attack_1",
  "target_stage": "stage3"
}
```
Returns attack vector generated by LLM reasoning.

### Explain Detection Result
```bash
POST /api/genai/explain-result
{
  "attack_vector": [...],
  "detected": false,
  "severity": "SUBTLE"
}
```
Returns LLM explanation of why attack succeeded/failed.
